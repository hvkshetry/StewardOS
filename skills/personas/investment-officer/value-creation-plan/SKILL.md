# Value Creation Plan

Design actionable levers to improve enterprise value in private holdings.

## MCP Tool Map

- Operating baseline context: `portfolio-monitoring` skill outputs (manual KPI inputs)
- Market/pricing context: `market-intel-direct.search_market_news`, `market-intel-direct.get_market_snapshot`
- Public benchmark context: `sec-edgar.sec_edgar_financial`

## Workflow

### Step 1: Baseline

- Define current revenue, EBITDA, margin, and cash conversion baseline.
- Identify largest value drag points.

### Step 2: Levers

- Revenue levers: pricing, mix, channel, retention.
- Cost levers: procurement, labor productivity, overhead, working capital.
- Strategic levers: product roadmap, capital structure, governance.

### Step 3: Plan Architecture

- 100-day actions.
- 12-month milestones.
- 24-month value bridge to target EBITDA and valuation.

### Step 4: Output

- Lever-by-lever action register.
- KPI owner and review cadence.
- Risks, dependencies, and contingency actions.
